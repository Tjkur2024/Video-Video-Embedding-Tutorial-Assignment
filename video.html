How to Embed a Video

Step 1: Locate a Video

Search for a video and click 'Share,' then 'Embed.'

Step 2: Copy the Embed Code

Copy the HTML code provided in the embed tab.

Step 3: Paste the Code in Your HTML

Open your HTML file, and paste the code in the <body> section.

Step 4: Save and Preview

Save your file and open it in a browser to see the embedded video.
